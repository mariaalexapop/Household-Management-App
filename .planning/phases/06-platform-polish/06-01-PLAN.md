```typescript
'use client'

import { useEffect } from 'react'

export function ServiceWorkerRegistration() {
  useEffect(() => {
    if ('serviceWorker' in navigator && process.env.NODE_ENV === 'production') {
      navigator.serviceWorker.register('/sw.js', { scope: '/', updateViaCache: 'none' })
    }
  }, [])
  return null
}
```

Note: Only register in production to avoid caching issues during development.

  <action>
Create placeholder app icons programmatically. Generate a simple "K" letter icon on the Kinship primary color (#5b76fe) background with white text.

**Option A (preferred): Use Node.js canvas to generate icons.**

Create a temporary script `scripts/generate-icons.mjs`:

```javascript
import { createCanvas } from 'canvas'
import fs from 'fs'
import path from 'path'

const sizes = [
  { name: 'icon-192x192.png', size: 192 },
  { name: 'icon-512x512.png', size: 512 },
  { name: 'icon-maskable.png', size: 512 },
  { name: 'apple-touch-icon.png', size: 180 },
]

const outDir = path.join(process.cwd(), 'public', 'icons')
fs.mkdirSync(outDir, { recursive: true })

for (const { name, size } of sizes) {
  const canvas = createCanvas(size, size)
  const ctx = canvas.getContext('2d')

  // Background
  ctx.fillStyle = '#5b76fe'
  if (name === 'icon-maskable.png') {
    // Maskable: fill entire canvas, letter in safe zone (center 80%)
    ctx.fillRect(0, 0, size, size)
  } else {
    // Regular: rounded corners approximation (fill entire, PNG transparency not needed for solid bg)
    ctx.fillRect(0, 0, size, size)
  }

  // Letter "K"
  ctx.fillStyle = '#ffffff'
  ctx.textAlign = 'center'
  ctx.textBaseline = 'middle'
  const fontSize = Math.round(size * 0.5)
  ctx.font = `bold ${fontSize}px sans-serif`
  ctx.fillText('K', size / 2, size / 2 + size * 0.02)

  const buffer = canvas.toBuffer('image/png')
  fs.writeFileSync(path.join(outDir, name), buffer)
  console.log(`Generated ${name} (${size}x${size})`)
}
```

Run: `node scripts/generate-icons.mjs`

**If canvas module is not available**, use an alternative approach:

Create minimal PNG files using the `sharp` library or a simple SVG-to-PNG conversion. If neither canvas nor sharp is available, create the icons using ImageMagick CLI:

```bash
mkdir -p public/icons
# Generate solid blue square with white K letter
for SIZE in 192 512 180; do
  convert -size ${SIZE}x${SIZE} xc:'#5b76fe' \
    -fill white -gravity center -pointsize $((SIZE/2)) \
    -font Helvetica-Bold -annotate +0+0 'K' \
    public/icons/icon-${SIZE}x${SIZE}.png
done
cp public/icons/icon-512x512.png public/icons/icon-maskable.png
cp public/icons/icon-180x180.png public/icons/apple-touch-icon.png
rm public/icons/icon-180x180.png
```

**For favicon.ico:** Convert the 192px icon to ICO format, or create a simple 32x32 and 16x16 ICO:

```bash
convert public/icons/icon-192x192.png -resize 32x32 public/icons/favicon.ico
```

If ImageMagick is not available, create placeholder PNGs using any available image tool. The icons are placeholder "K" logos meant to be replaced with a designed icon later.

**After generating icons:** Copy favicon.ico to the project root as well (Next.js serves `app/favicon.ico` or `public/favicon.ico`):

```bash
cp public/icons/favicon.ico public/favicon.ico
```
