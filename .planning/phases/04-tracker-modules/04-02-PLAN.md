**src/app/actions/documents.ts** -- Create 'use server' file with 4 server actions following the kids.ts pattern exactly (auth check via supabase.auth.getUser(), getMemberRow for household, Zod validation):

1. `getUploadUrl(data: unknown)` -- Validates: module ('insurance' | 'electronics'), entityId (uuid), fileName (string, max 255), fileSizeBytes (number, max 10485760). Returns `{ signedUrl, token, path }`. Storage path convention: `{householdId}/{module}/{entityId}/{fileName}`. Uses `supabase.storage.from('documents').createSignedUploadUrl(storagePath)`.

2. `confirmUpload(data: unknown)` -- Called after client-side upload completes. Validates: module, entityId, documentType ('policy' | 'warranty' | 'manual'), fileName, storagePath, fileSizeBytes. Inserts a row into the `documents` table. Returns `{ id }`. Revalidates the module path (e.g., '/insurance' or '/electronics').

3. `getDownloadUrl(data: unknown)` -- Validates: documentId (uuid). Looks up the document row, verifies household membership, creates a signed download URL with 1-hour expiry via `supabase.storage.from('documents').createSignedUrl(storagePath, 3600)`. Returns `{ url }`.

4. `deleteDocument(data: unknown)` -- Validates: documentId (uuid). Looks up the document row, verifies household membership, deletes the file from Supabase Storage via `supabase.storage.from('documents').remove([storagePath])`, then deletes the DB row. Returns `{ success: true }`. Revalidates the module path.

Import the `ActionResult` type from kids.ts or redefine it locally (same interface). Import `documents` table from schema. Import `getMemberRow` helper or redefine it locally (same pattern).

export function formatCostFromCents(cents: number | null | undefined): string {
  if (cents == null) return '--'
  return new Intl.NumberFormat('en-GB', { style: 'currency', currency: 'GBP' }).format(cents / 100)
}

export function centsToPounds(cents: number): number {
  return cents / 100
}

export function poundsToCents(pounds: number): number {
  return Math.round(pounds * 100)
}
